Each transfer can have a compact or detailed response.<br/>
It depends on the parameter ``compact``.
* Fix required fields are
    * surveyDate
    * id
    * parent_id
    * profile
    * mandator
    * state
    * controllerId
    * _operation
    * source
    	* host
    * target
    	* host
    	* protocol
    * numOfFiles
    * start
    * end
* The detailed view has the following optional fields in addition to the compact view
    * source
    	* user
    	* port
    	* protocol
    * target
    	* user
    	* port
    * jump
    	* host
    	* user
    	* port
    	* protocol
    * jobChain
    * order
    * job
