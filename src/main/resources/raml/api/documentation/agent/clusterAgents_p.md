Contains collection of Cluster Agents where each Agent following fields
* controllerId
* agentId
* agentName
* agentNameAliases
* subagents where each Subagent has following fields
	* subagentId
	* url
	* title (optional)
	* deployed
	* disabled
	* isDirector (NO_DIRECTOR, PRIMARY_DIRECTOR, SECONDARY_DIRECTOR)
	* isClusterWatcher
