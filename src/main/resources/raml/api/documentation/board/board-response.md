Each board can have a compact or detailed response.<br/>
It depends on the parameter ``compact``.<br/>
* The fields are
    * path
    * numOfNotices
    * numOfExpectingOrders
    * numOfPostedNotices
    * numOfExpectedNotices
    * numOfAnnouncements
    * notices (compact==false)
        * id
        * state (POSTED, EXPECTED, ANNOUNCED)
        * expectingOrders
