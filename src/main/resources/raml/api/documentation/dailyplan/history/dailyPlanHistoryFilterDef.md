All filter elements will be combined with "and".  The values in the controllerIds element will be combined with "or".
If dateTo is not set then entries for dateFrom will be considered.

The filter elements are

<ul>

controllerId
</li>

controllerIds

<li>
dateFrom
</li>
<li>
dateTo
</li>

</li>
<li>
